# Feature Importance Analysis for CICIDS2017

Code source du projet de recherche sur la stabilité des features pour la détection d'intrusions.
Ce projet utilise une approche par fenêtre glissante avec Random Forest, SelectKBest et Permutation Importance.

## Structure

- `src/` : Contient le script principal d'analyse.
- `data/` : Dossier pour le dataset (le fichier CSV n'est pas inclus dans ce dépôt).
- `results/` : Dossier où les graphiques et résultats seront générés.

## Installation

1. Cloner le dépôt :
   ```bash
   git clone [https://github.com/TON_PSEUDO/cicids2017-feature-analysis.git](https://github.com/TON_PSEUDO/cicids2017-feature-analysis.git)
   cd cicids2017-feature-analysis
   ```
